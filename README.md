### Gmail email check

``` python
from EmailChecker import SidraELEzz

email = "<your email>"

Logan = SidraELEzz.Gmail(str(email))

if Logan ==True:
	print ("Email is available")
	
elif Logan ==False:
	print("Email is not available")

	
```
